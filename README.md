# Image Sharing API

API that allows 2 users to upload, sort and share

## Installation

In the root directory

```bash
  yarn
  yarn compile:dev
  yarn start:dev
```

1. Create .env file(see .envexample)

2. Start the application

   In the root directory

   ```
   yarn compile
   yarn start
   ```
